# hangman-game
A beginner-friendly Hangman game implemented in C++ and Python. The game runs in the terminal where players guess letters to reveal the hidden word before running out of tries.
This project was created as a beginner programming exercise to practice basic concepts like loops, conditionals, strings, and vectors/lists.

🛠 Languages Used
C++
Python

📚 Concepts Practiced
Random word selection
Loops
Conditional statements
Strings
Vectors (C++)
Lists (Python)
Basic game logic
